# Collabify

**Collabify** is a modern team collaboration platform designed to bring communication, task management and scheduling together in one place. It helps teams stay aligned, improve productivity and work smoothly with real-time updates and clean interfaces. Built with the MERN stack, Collabify gives users a simple way to chat, manage tasks, plan schedules and stay organized.

## üåüFeatures

- **Real-Time Communication**: Chat with your team through channels and message threads that sync instantly.
- **Task Management**: Create tasks, assign them, set deadlines and track progress in a structured way.
- **Project Organization**: Group tasks into projects for clear roadmap planning and better workflow tracking.
- **Scheduling & Calendar**: Built-in scheduling tools make it easy to plan meetings or track important dates.
- **Smart Dashboard**: Clean overview of chats, tasks and upcoming events.
- **Secure Authentication**: Users sign up and log in through a secure JWT-based system.
- **Responsive Design**: Works seamlessly across desktop, tablet, and mobile devices.
- **Responsive UI**: The interface adjusts smoothly across desktops, tablets and mobile devices.
- **Optimized Dark Theme**: A polished, night-gradient layout for a focused and immersive work environment.

## üöÄInstallation

Follow the steps below to run the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/Collabify.git
   
2. **Navigate to the project directory**
   ```bash
   cd Collabify

3. **Install dependencies**
   ### Backend and Frontend
   ```bash
   cd backend
   cd frontend
   npm install

4. **Setup Enviornment Variables**
   ### Create a .env file in the backend directory and add:**
   ```bash
   PORT=8000
   MONGO_URL=your-mongodb-connection-string
   JWT_SECRET=your-jwt-secret
   EMAIL_SERVER = smtp.gmail.com
   EMAIL_PORT = 587
   EMAIL_USERNAME = your-email@example.com
   EMAIL_PASSWORD = your-email-app-password
   EMAIL_FROM = "Project Collaboration Team <team@yourdomain.com>"
   GOOGLE_CLIENT_ID = your-google-client-id
   GOOGLE_CLIENT_SECRET = your-google-client-secret


5. **Run the backend**
   ```bash
   cd backend
   npm start
   
6. **Run the frontend**
   ```bash
   cd frontend
   npm start
